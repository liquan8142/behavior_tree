🚀 如何安装完整的GitBook环境

要获得完整的GitBook渲染效果，请运行：

# 1. 安装GitBook CLI
npm install -g gitbook-cli

# 2. 进入文档目录
cd /home/liquan/.openclaw/workspace/btcpp_website/behavior_tree

# 3. 安装依赖
npm install
gitbook install

# 4. 启动本地服务器
gitbook serve
# 访问: http://localhost:4000

或者使用提供的脚本：

./start-local.sh   # 启动开发服务器
./build.sh        # 构建静态网站
./preview.sh      # 预览已构建的网站
